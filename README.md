🤖 MyBot

MyBot is a simple AI chatbot built using HTML, CSS, and JavaScript. It uses the Google Gemini API to generate real-time responses through a clean and responsive chat interface.

Features-----
- AI-powered chatbot
- Responsive design
- Dark & Light mode
- Emoji picker
- Typing animation
- Modern UI

Tech Stack-----
- HTML5
- CSS3
- JavaScript (ES6)
- Google Gemini API
- Git & GitHub

Installation----
1. Clone the repository.
2. Open `script.js`.
3. Replace `YOUR_API_KEY_HERE` with your Gemini API key.
4. Open `index.html` in your browser.
